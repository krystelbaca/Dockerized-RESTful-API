# dockerized-restful-api

Dockerized RESTful API

Side project: dockerized application as a microservice. 
Author: Krystel Baca Salayandia. 

# Local setup

## Pre-requisites:

1. Docker installed

## Launch the app

Start the API:

```sh
npm start
```

Development mode:

```sh
npm run dev
```

# Test

```sh
npm test
```
